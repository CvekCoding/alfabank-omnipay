# Download

## Composer 

```
// This assumes that you have composer installed globally
composer require CvekCoding/alfabank-omnipay
```

## Solving problems with minimal stability

Add to your composer.json

```json
{
  "minimum-stability":"dev",
  "prefer-stable": true
}

```
